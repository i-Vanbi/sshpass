# SSHPass

SSHpass offers you the ability to automatically offer a password via SSH when
you are prompted for it.

## Install

```bash
git clone git://github.com/kevinburke/sshpass.git
cd sshpass
./configure
make && make install
```
## 用法说明  
1：直接远程连接某台主机：  
`sshpass -p xxx ssh root@192.168.11.11`xxx为密码。
